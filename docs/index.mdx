`coding-mcp` is a multi-project MCP server for coding agents. It is built to support remote coding workflows where agents can inspect, edit, test, and ship changes without cloning source code to the agent machine or setting up each repository locally.

It provides safe file and git automation, consistent structured responses, HTTP/STDIO transports, API-key RBAC, and OpenTelemetry hooks.

Use this documentation to set up the server, understand architecture, and extend tools safely.

## Core capabilities

- Multi-root project discovery and persistent registry
- Safe file CRUD and patching with unified diff hunk support
- Git operations with structured output
- Allowlist-based command execution
- Binary-safe file resources for text and binary payloads
- HTTP auth with role-based access control
- OpenTelemetry hooks for requests and tool operations
