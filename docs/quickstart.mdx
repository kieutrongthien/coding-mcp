<Info>
  **Prerequisites**:
  - Node.js 20 or newer
  - At least one folder containing project repositories (for example `/projects`)
</Info>

## Install

Install globally from npm:

```bash
npm install -g @kieutrongthien/coding-mcp
```

Or run directly with npx:

```bash
npx @kieutrongthien/coding-mcp serve --transport http --host 0.0.0.0 --port 4000 --mode streamable
```

If you want to develop from source:

```bash
git clone https://github.com/kieutrongthien/coding-mcp.git
cd coding-mcp
npm install
npm run build
```

## Configure environment

Create a local env file from the example:

```bash
cp .env.example .env
```

Set project roots in `.env`:

```bash
PROJECTS_ROOTS=/projects,/srv/repos
```

## Initialize the project registry

Use the CLI to scan and persist projects from a root:

```bash
coding-mcp init /projects
```

Add or remove roots later:

```bash
coding-mcp add /srv/repos
coding-mcp remove /srv/repos
```

## Start the server

HTTP mode:

```bash
coding-mcp serve --transport http --host 0.0.0.0 --port 4000 --mode streamable
```

STDIO mode:

```bash
coding-mcp serve --transport stdio
```

## Verify

Run quality checks after setup:

```bash
npm run typecheck
npm test
```
