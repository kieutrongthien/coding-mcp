`coding-mcp` là MCP server đa dự án cho coding agent. Dự án được tạo ra để hỗ trợ workflow xây dựng agent code từ xa, không cần clone source code về máy chạy agent và không cần setup từng repository tại máy đó.

Hệ thống cung cấp tự động hóa file và git an toàn, response có cấu trúc thống nhất, transport HTTP/STDIO, API-key RBAC và OpenTelemetry hooks.

Hãy dùng bộ tài liệu này để thiết lập server, hiểu kiến trúc, và mở rộng tools một cách an toàn.

## Khả năng cốt lõi

- Khám phá nhiều root dự án và lưu registry bền vững
- File CRUD an toàn và patching hỗ trợ unified diff hunk
- Thao tác Git với output có cấu trúc
- Thực thi command theo allowlist
- File resources an toàn cho text và binary
- HTTP auth với role-based access control
- OpenTelemetry hooks cho request và tool operations
