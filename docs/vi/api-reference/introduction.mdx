title: "Tổng quan giao diện MCP"
description: "Tham chiếu đầy đủ cho tools, resources và prompts giúp agent code trên repository từ xa."
---

Giao diện này được thiết kế để agent thao tác trên repository từ xa, không cần clone source code về máy chạy agent.

Server cung cấp năng lực MCP theo 3 nhóm:

| Nhóm | Mô tả |
|------|-------|
| **Tools** | Thao tác chủ động: quản lý project, chỉnh sửa file, tìm kiếm, git và thực thi command |
| **Resources** | Dữ liệu read-only thụ động: cây dự án, nội dung file, metadata và git status |
| **Prompts** | Hướng dẫn workflow có sẵn: implement, analyze, review, generate commit, fix tests |

## Response envelope

Tất cả tool responses đều dùng chung định dạng envelope nhất quán.

Thao tác thành công:

```json
{
  "ok": true,
  "operation": "read_file",
  "project_id": "abc123",
  "data": {},
  "warnings": [],
  "errors": [],
  "request_id": "uuid",
  "duration_ms": 12
}
```

Thao tác thất bại:

```json
{
  "ok": false,
  "operation": "write_file",
  "error_code": "SECURITY_ERROR",
  "message": "Operation blocked for protected path",
  "details": {}
}
```

## Vị trí triển khai handlers

- `src/mcp/tools/*` — tool handlers
- `src/mcp/resources/*` — resource handlers
- `src/mcp/prompts/*` — prompt handlers

Dùng các trang tham chiếu trong phần này để xem toàn bộ bề mặt năng lực đã hỗ trợ.
