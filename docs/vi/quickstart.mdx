`coding-mcp` được ưu tiên cho tình huống agent cần làm việc từ xa trên repository, không cần clone source code hoặc setup từng repo trên máy chạy agent.

<Info>
  **Điều kiện tiên quyết**:
  - Node.js 20 trở lên
  - Ít nhất một thư mục chứa repository (ví dụ `/projects`)
</Info>

## Cài đặt

Cài đặt global từ npm:

```bash
npm install -g @kieutrongthien/coding-mcp
```

## Khởi tạo project registry

Dùng CLI để quét và lưu toàn bộ dự án trong một thư mục:

```bash
cd /my-projects
coding-mcp init
```

hoặc thêm các dự án từ một thư mục khác:

```bash
coding-mcp init /other-projects
```

Thêm hoặc xóa dự án:

```bash
coding-mcp add /projects/my-repo
coding-mcp remove /projects/my-repo
```

## Khởi động server

HTTP mode:

```bash
coding-mcp serve --transport http --host 0.0.0.0 --port 4000 --mode streamable
```

STDIO mode:

```bash
coding-mcp serve --transport stdio
```
