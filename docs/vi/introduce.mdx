`coding-mcp` là MCP server production-ready được xây dựng với một mục tiêu chính: **cho phép AI agent viết và ship code trên repository từ xa mà không cần clone về máy chạy agent**.

Triển khai server một lần trên máy chứa code. Bất kỳ agent tương thích MCP nào — Claude, Cursor, hay bất kỳ HTTP client nào — đều có thể kết nối và ngay lập tức truy cập các tool thao tác file, git, tìm kiếm, build và test trên toàn bộ project đã index.

```
┌─────────────────────────────────────────────┐
│  AI Agent (Claude / Cursor / any MCP client) │
└────────────────┬────────────────────────────┘
                 │  MCP qua HTTP hoặc STDIO
                 ▼
┌─────────────────────────────────────────────┐
│           coding-mcp server                  │
│  (chạy trên máy chứa repositories)          │
│                                              │
│  Tools: files · git · search · commands      │
│  Resources: project index · file content     │
│  Prompts: implement · review · fix tests     │
└──────────────────┬──────────────────────────┘
                   │  filesystem + git
                   ▼
        /projects/repo-a  /projects/repo-b  …
```

## Khả năng cốt lõi

| Khả năng | Mô tả |
|----------|-------|
| Multi-root project registry | Quét một hoặc nhiều thư mục gốc và lưu index dự án để agent truy vấn qua ID ổn định |
| Thao tác file | Đọc, ghi, tạo, xóa, di chuyển file và apply unified diff hoặc structured edits |
| Git tự động hóa | Status, diff, branch, commit, push, tag — toàn bộ với output JSON có cấu trúc |
| Tìm kiếm và phân tích | Tìm theo tên file, grep nội dung bằng regex, cây dự án và tóm tắt stack |
| Thực thi command an toàn | Build, test và lint tự động phát hiện từ tooling dự án; lệnh tùy ý qua allowlist |
| File resources an toàn cho binary | File được phục vụ dưới dạng `text` hoặc base64 `blob` theo MIME type |
| HTTP auth và RBAC | Xác thực bằng API key với vai trò `viewer`, `editor` và `admin` |
| OpenTelemetry hooks | Spans cho mỗi HTTP request và tool operation |
